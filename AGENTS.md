# AGENTS.md

## Sobre o projeto
Este projeto se chama Retorno Fácil.
Ele é um mini sistema para organizar clientes e retornos de atendimento via WhatsApp.

## Como você deve trabalhar neste projeto
- Explique as mudanças de forma simples e didática.
- Antes de alterar os arquivos, diga em poucas linhas o que pretende fazer.
- Faça mudanças pequenas e fáceis de revisar.
- Não adicionar bibliotecas externas sem necessidade.
- Preferir HTML, CSS e JavaScript puro.
- Manter o código simples, legível e organizado.
- Atualizar o README quando a funcionalidade mudar.
- Evitar complexidade desnecessária.

## Estilo de código
- HTML semântico
- CSS organizado por seções
- JavaScript claro, com nomes fáceis de entender
- Comentários apenas quando realmente ajudarem

## Definição de pronto
Uma tarefa só está pronta quando:
- a funcionalidade funciona
- o código continua simples
- o projeto continua organizado
- o README está coerente com o que foi feito
